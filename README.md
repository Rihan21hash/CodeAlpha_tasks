# 1. E-Commerce Website

A modern and responsive e-commerce web application designed to provide users with a seamless online shopping experience. The platform allows users to browse products, explore product details, and manage shopping activities efficiently.

## Features

- Responsive design
- Product listing
- Product detail pages
- Category filtering
- Search functionality
- Shopping cart system
- Clean UI

## Key Functionalities

### Product Browsing
Users can explore products in an organized layout.

### Product Search and Filtering
Products can be searched and filtered by categories.

### Cart Management
Users can add or remove products from the cart.

# 2. Text Share Social Media App

A text-based social media application that allows users to share posts, express thoughts, and engage with content in a clean and interactive environment.

## Features

- Create text posts
- Dynamic feed
- Responsive UI
- Real-time content updates
- User interaction system

## Key Functionalities

### Create Posts
Users can instantly create and share text posts.

### Real-Time Feed
Shared posts are dynamically displayed in the feed.

### User Interaction
Users can engage with posts through interactions.
